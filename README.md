### :rotating_light: *Important*

Add your google map key :key: here : src\app\app.module.ts | Replace : {{Your API Key}} 
```
    AgmCoreModule.forRoot({
      apiKey: '{{Your API Key}}'
    })
```

## 	:wink: Run on Local

```
- npm install
- npm start
```

### :page_facing_up: Packages

- Bootstrap 4 
- Angular Google Maps (AGM) https://angular-maps.com/
